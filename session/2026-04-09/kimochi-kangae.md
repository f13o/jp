## 気持 (きも)ち・考 (かんが)え -- Sentimientos y pensamiento

```sentence-n
大事 (だいじ)な 事 (こと)です。
Es algo importante.
```

```sentence-n
大事 (だいじ)だと 思 (おも)います。
Creo que es importante.
```

```sentence-n
意見 (いけん)を 聞 (き)きます。
Escucho opiniones.
```

```sentence-n
動物 (どうぶつ)が 好 (す)きです。
Me gustan los animales.
```

```sentence-n
生 (せい)と 死 (し)を 考 (かんが)えます。
Pienso en la vida y la muerte.
```

```sentence-n
強 (つよ)い 気持 (きも)ちがあります。
Tengo sentimientos fuertes.
```

```sentence-n
よく 考 (かんが)えます。
Lo pienso bien.
```

```sentence-n
何 (なに)を 考 (かんが)えていますか。
¿En qué estás pensando?
```

```sentence-n
これは 特別 (とくべつ)な 日 (ひ)です。
Este es un día especial.
```

```sentence-n
運動 (うんどう)が 好 (す)きです。
Me gusta hacer ejercicio.
```

```sentence-n
大切 (たいせつ)な 人 (ひと)がいます。
Hay alguien importante para mí.
```

```sentence-n
音楽 (おんがく)が 好 (す)きです。
Me gusta la música.
```

```sentence-n
二人 (ふたり)の 意見 (いけん)が 合 (あ)います。
Las opiniones de los dos coinciden.
```

```sentence-n
絵 (え)を 描 (か)くのが 好 (す)きだ。
Me gusta dibujar.
```

```sentence-n
この 絵 (え)は 有名 (ゆうめい)です。
Esta pintura es famosa.
```
