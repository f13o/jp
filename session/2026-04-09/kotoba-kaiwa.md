## 言葉 (ことば)・会話 (かいわ) -- Palabras y conversacion

```sentence-n
音楽 (おんがく)を 聞 (き)きます。
Escucho música.
```

```sentence-n
何 (なん)と 言 (い)いましたか。
¿Qué dijiste?
```

```sentence-n
理由 (りゆう)を 聞 (き)きます。
Pregunto la razón.
```

```sentence-n
用意 (ようい)ができました。
Ya está listo.
```

```sentence-n
用事 (ようじ)があります。
Tengo algo que hacer.
```

```sentence-n
この 言葉 (ことば)を 使 (つか)います。
Uso esta palabra.
```

```sentence-n
計画 (けいかく)を 立 (た)てます。
Hago un plan.
```

```sentence-n
いい 経験 (けいけん)でした。
Fue una buena experiencia.
```

```sentence-n
注意 (ちゅうい)してください。
Ten cuidado, por favor.
```

```sentence-n
公 (おおやけ)の 場 (ば)で 話 (はな)します。
Hablo en un lugar público.
```
