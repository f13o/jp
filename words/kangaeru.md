---
writing: 考える
pronunciation: かんがえる
type: verbo ichidan (grupo II)
---

## Significado

- Pensar, reflexionar
- Considerar, tener en cuenta

## Conjugaciones

Raiz: **考え** (かんがえ)

| Forma           | Conjugacion      |
| --------------- | ---------------- |
| Diccionario     | 考える           |
| Masu            | 考えます         |
| Negativa        | 考えない         |
| Te              | 考えて           |
| Pasado          | 考えた           |
| Pasado negativo | 考えなかった     |
| Potencial       | 考えられる       |
| Volitiva        | 考えよう         |

## Uso

{よく 考 (かんが) えます。} Pienso bien.

{何 (なに) を 考 (かんが) えた？} ¿Que pensaste?

{考 (かんが) えてください。} Por favor, pienselo.
