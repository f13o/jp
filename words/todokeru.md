---
writing: 届ける
pronunciation: とどける
type: verbo ichidan (grupo II)
---

## Significado

- Entregar, hacer llegar
- Reportar, notificar (a las autoridades)

## Conjugaciones

Raiz: **届け** (とどけ)

| Forma           | Conjugacion      |
| --------------- | ---------------- |
| Diccionario     | 届ける           |
| Masu            | 届けます         |
| Negativa        | 届けない         |
| Te              | 届けて           |
| Pasado          | 届けた           |
| Pasado negativo | 届けなかった     |
| Potencial       | 届けられる       |
| Volitiva        | 届けよう         |

## Uso

{荷物 (にもつ) を 届 (とど) けます。} Entrego el paquete.

{手紙 (てがみ) を 届 (とど) けた。} Entregue la carta.

{届 (とど) けてください。} Por favor, entreguelo.
