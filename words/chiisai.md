writing: 小さい

## Significado

- Pequeño/a

## Conjugaciones

Raiz: **小さ** (ちいさ)

| Forma           | Conjugacion      |
| --------------- | ---------------- |
| Presente        | 小さい           |
| Negativa        | 小さくない       |
| Pasado          | 小さかった       |
| Pasado negativo | 小さくなかった   |
| Adverbio        | 小さく           |

## Uso

{小 (ちい) さい 犬 (いぬ) がいる。} Hay un perro pequeño.

{この 部屋 (へや) は 小 (ちい) さい。} Esta habitacion es pequeña.

{星 (ほし) が 小 (ちい) さく 見 (み) える。} Las estrellas se ven pequeñas.
