## Significado

- Beber, tomar

## Conjugaciones

Raiz: **飲** (の)

| Forma           | Conjugacion          |
| --------------- | -------------------- |
| Diccionario     | 飲む (のむ)           |
| Masu            | 飲みます (のみます)     |
| Negativa        | 飲まない (のまない)     |
| Te              | 飲んで (のんで)        |
| Pasado          | 飲んだ (のんだ)        |
| Pasado negativo | 飲まなかった (のまなかった) |
| Potencial       | 飲める (のめる)        |
| Volitiva        | 飲もう (のもう)        |

## Uso

{水 (みず) を 飲 (の) む。} Beber agua.

{お 茶 (ちゃ) を 飲 (の) みます。} Tomo te.

{毎朝 (まいあさ) コーヒーを 飲 (の) んでいます。} Todas las mananas tomo cafe.
