## Significado

- Beber, tomar

## Conjugaciones

Raiz: **飲** (の)

| Forma           | Conjugacion          |
| --------------- | -------------------- |
| Diccionario     | 飲む (のむ)           |
| Masu            | 飲みます (のみます)     |
| Negativa        | 飲まない (のまない)     |
| Te              | 飲んで (のんで)        |
| Pasado          | 飲んだ (のんだ)        |
| Pasado negativo | 飲まなかった (のまなかった) |
| Potencial       | 飲める (のめる)        |
| Volitiva        | 飲もう (のもう)        |

## Uso

水を飲む。 みずをのむ。 Beber agua.

お茶を飲みます。 おちゃをのみます。 Tomo te.

毎朝コーヒーを飲んでいます。 まいあさコーヒーをのんでいます。 Todas las mananas tomo cafe.
