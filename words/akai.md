writing: 赤い
pronunciation: あかい
type: adjetivo-i

## Significado

- Rojo/a

## Conjugaciones

Raiz: **赤** (あか)

| Forma           | Conjugacion    |
| --------------- | -------------- |
| Presente        | 赤い           |
| Negativa        | 赤くない       |
| Pasado          | 赤かった       |
| Pasado negativo | 赤くなかった   |
| Adverbio        | 赤く           |

## Uso

{赤 (あか) い 花 (はな) が 好 (す) きです。} Me gustan las flores rojas.

{あの 車 (くるま) は 赤 (あか) い。} Ese coche es rojo.

{空 (そら) が 赤 (あか) くなった。} El cielo se puso rojo.
