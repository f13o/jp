---
writing: 始める
pronunciation: はじめる
type: verbo ichidan (grupo II)
---

## Significado

- Empezar, comenzar (algo)
- Iniciar

## Conjugaciones

Raiz: **始め** (はじめ)

| Forma           | Conjugacion  |
| --------------- | ------------ |
| Diccionario     | 始める       |
| Masu            | 始めます     |
| Negativa        | 始めない     |
| Te              | 始めて       |
| Pasado          | 始めた       |
| Pasado negativo | 始めなかった |
| Potencial       | 始められる   |
| Volitiva        | 始めよう     |

## Uso

仕事を始めます。 しごとをはじめます。 Empiezo el trabajo.

始めてください。 はじめてください。 Por favor, empiece.

昨日始めた。 きのうはじめた。 Empece ayer.
