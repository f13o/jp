---
writing: 始める
pronunciation: はじめる
type: verbo ichidan (grupo II)
---

## Significado

- Empezar, comenzar (algo)
- Iniciar

## Conjugaciones

Raiz: **始め** (はじめ)

| Forma           | Conjugacion  |
| --------------- | ------------ |
| Diccionario     | 始める       |
| Masu            | 始めます     |
| Negativa        | 始めない     |
| Te              | 始めて       |
| Pasado          | 始めた       |
| Pasado negativo | 始めなかった |
| Potencial       | 始められる   |
| Volitiva        | 始めよう     |

## Uso

{仕事 (しごと) を 始 (はじ) めます。} Empiezo el trabajo.

{始 (はじ) めてください。} Por favor, empiece.

{昨日 (きのう) 始 (はじ) めた。} Empece ayer.
