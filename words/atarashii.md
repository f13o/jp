---
writing: 新しい
pronunciation: あたらしい
type: adjetivo-i
---

## Significado

- Nuevo/a
- Reciente, fresco

## Conjugaciones

Raiz: **新し** (あたらし)

| Forma           | Conjugacion      |
| --------------- | ---------------- |
| Presente        | 新しい           |
| Negativa        | 新しくない       |
| Pasado          | 新しかった       |
| Pasado negativo | 新しくなかった   |
| Adverbio        | 新しく           |

## Uso

{新 (あたら) しい 靴 (くつ) を 買 (か) った。} Compre zapatos nuevos.

{新 (あたら) しい 先生 (せんせい) はとても 優 (やさ) しい。} La nueva profesora es muy amable.

{何 (なに) か 新 (あたら) しいことを 始 (はじ) めたい。} Quiero empezar algo nuevo.
