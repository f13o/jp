# 動詞 (どうし) -- Verbos

Los verbos en japones se conjugan cambiando su terminacion. Todos los verbos en forma
diccionario terminan en un sonido de la columna う (u, ku, su, tsu, nu, bu, mu, ru).

## Tres grupos

### Grupo 1 -- 五段 (ごだん)

La mayoria de los verbos. Su raiz cambia entre las cinco filas de la tabla de kana (por
eso "cinco escalones"). Terminan en consonante + う.

- 飲 (の)む -- beber
- 書 (か)く -- escribir
- 話 (はな)す -- hablar
- 待 (ま)つ -- esperar
- 読 (よ)む -- leer
- 行 (い)く -- ir

### Grupo 2 -- 一段 (いちだん)

Siempre terminan en る, y antes de る hay un sonido de la columna い o え. La raiz no cambia
-- solo se quita る y se agrega la terminacion.

- 食 (た)べる -- comer
- 見 (み)る -- ver
- 起 (お)きる -- levantarse, despertarse
- 寝 (ね)る -- dormir

### Grupo 3 -- 不規則 (ふきそく) -- irregulares

Solo dos:

- する -- hacer
- 来 (く)る -- venir

## Forma ます -- cortesia

La forma ます es la forma cortes del presente/futuro. Es la primera que se aprende porque
es la mas usada en conversacion.

### Como formar ます

**Grupo 1:** cambiar la ultima silaba a la fila い del 五十音 (ごじゅうおん) y agregar ます.

| Verbo       | Raiz い     | Forma ます      |
| ----------- | ----------- | --------------- |
| 飲 (の)む   | 飲 (の)み   | 飲 (の)みます   |
| 書 (か)く   | 書 (か)き   | 書 (か)きます   |
| 話 (はな)す | 話 (はな)し | 話 (はな)します |
| 待 (ま)つ   | 待 (ま)ち   | 待 (ま)ちます   |
| 読 (よ)む   | 読 (よ)み   | 読 (よ)みます   |

**Grupo 2:** quitar る y agregar ます.

| Verbo       | Forma ます    |
| ----------- | ------------- |
| 食 (た)べる | 食 (た)べます |
| 見 (み)る   | 見 (み)ます   |
| 起 (お)きる | 起 (お)きます |

**Grupo 3:**

| Verbo     | Forma ます  |
| --------- | ----------- |
| する      | します      |
| 来 (く)る | 来 (き)ます |

## Conjugaciones de ます

| Forma        | Significado       | Ejemplo                          |
| ------------ | ----------------- | -------------------------------- |
| ます         | presente / futuro | 飲 (の)みます -- bebo / bebere   |
| ません       | negativo          | 飲 (の)みません -- no bebo       |
| ました       | pasado            | 飲 (の)みました -- bebi          |
| ませんでした | pasado negativo   | 飲 (の)みませんでした -- no bebi |

## Ejemplos

```sentence-n
まいにちみずをのみます
todos los dias bebo agua
毎日水を飲みます
```

```sentence-n
にほんごをべんきょうします
estudio japones
日本語を勉強します
```

```sentence-n
あしたがっこうにいきます
mañana voy a la escuela
明日学校に行きます
```

```sentence-n
きのうほんをよみました
ayer lei un libro
昨日本を読みました
```

```sentence-n
あさごはんをたべません
no desayuno
朝ご飯を食べません
```

```sentence-n
きのうえいがをみませんでした
ayer no vi la pelicula
昨日映画を見ませんでした
```
