# 代名詞 (だいめいし) - Pronombres y contexto

En japones los pronombres existen pero se usan mucho menos que en espanol. El verbo no
cambia por persona: 読 (よ) む es "leer" sin importar quien lee. La persona se entiende
por contexto.

## Omision del sujeto

En espanol "leo" ya implica "yo" por la conjugacion. En japones 読 (よ) む no indica
persona -- se infiere de la situacion. Agregar el pronombre todo el tiempo suena
antinatural.

```sentence-n
ほんをよむ
leo un libro (o: lees / lee / leemos...)
本を読む
```

```sentence-n
みずをのむ
bebo agua (o: bebes / bebe...)
水を飲む
```

```sentence-n
にほんごをはなす
hablo japones (o: hablas / habla...)
日本語を話す
```

Solo cuando hay ambiguedad se agrega el pronombre + は:

```sentence-n
わたしはほんをよむ
yo leo un libro
私は本を読む
```

```sentence-n
かれはみずをのむ
el bebe agua
彼は水を飲む
```

```sentence-n
かのじょはにほんごをはなす
ella habla japones
彼女は日本語を話す
```

## 一人称 (いちにんしょう) - primera persona

| pronombre | lectura | uso |
|---|---|---|
| 私 | わたし | neutro, sirve en cualquier situacion |
| 私 | わたくし | muy formal |
| 僕 | ぼく | masculino, casual |
| 俺 | おれ | masculino, muy informal |
| あたし | -- | femenino, casual |

私 (わたし) es el mas seguro. Los demas dependen del genero y la formalidad.

```sentence-n
わたしはがくせいです
yo soy estudiante
私は学生です
```

```sentence-n
ぼくはにほんにいく
yo voy a Japon
僕は日本に行く
```

## 二人称 (ににんしょう) - segunda persona

| pronombre | lectura | uso |
|---|---|---|
| あなた | -- | tu (pero se evita) |
| 君 | きみ | tu (casual, de superior a inferior) |
| お前 | おまえ | tu (muy informal, puede sonar rudo) |

En japones **se evita decir "tu"** directamente. Es mas natural usar el nombre de la
persona + さん.

```sentence-n
たなかさんはがくせいですか
Tanaka, eres estudiante?
田中さんは学生ですか
```

```sentence-n
やまださんはにほんじんですか
Yamada, eres japones?
山田さんは日本人ですか
```

```sentence-n
せんせいはどこにいきますか
profesor, a donde va?
先生はどこに行きますか
```

En vez de あなたは学生ですか ("tu eres estudiante?"), se dice 田中 (たなか) さんは学生 (がくせい) ですか usando el nombre.

## 三人称 (さんにんしょう) - tercera persona

| pronombre | lectura | uso |
|---|---|---|
| 彼 | かれ | el |
| 彼女 | かのじょ | ella |

Tambien se usan poco. Es mas comun referirse a la persona por su nombre.

```sentence-n
かれはせんせいです
el es profesor
彼は先生です
```

```sentence-n
かのじょはにほんにすんでいる
ella vive en Japon
彼女は日本に住んでいる
```

```sentence-n
たなかさんはせんせいです
Tanaka es profesor
田中さんは先生です
```

## 複数形 (ふくすうけい) - plural

Se agrega たち para hacer plural.

| singular | plural | lectura |
|---|---|---|
| 私 | 私たち | わたしたち |
| 僕 | 僕たち | ぼくたち |
| 彼 | 彼ら | かれら |
| 彼女 | 彼女たち | かのじょたち |

```sentence-n
わたしたちはがくせいです
nosotros somos estudiantes
私たちは学生です
```

```sentence-n
かれらはにほんにいく
ellos van a Japon
彼らは日本に行く
```

```sentence-n
ぼくたちはともだちです
nosotros somos amigos
僕たちは友達です
```

## Resumen

La regla principal: **si se entiende por contexto, no se dice**. Los pronombres en
japones son opcionales, no obligatorios. Usarlos de mas suena redundante o incluso
descortes (especialmente あなた).
