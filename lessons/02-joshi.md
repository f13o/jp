# 助詞 (じょし) -- Particulas

Las particulas son palabras cortas que van despues de un sustantivo e indican su funcion
en la oracion. No tienen significado propio, solo conectan.

## は -- tema

Se lee "wa" (no "ha") cuando es particula. Marca el tema: "en cuanto a..."

```sentence-n
ねこはどうぶつです
el gato es un animal
猫は動物です
```

```sentence-n
やまはたかいです
la montaña es alta
山は高いです
```

## を -- objeto directo

Se lee "o". Marca lo que recibe la accion del verbo.

```sentence-n
みずをのむ
bebo agua
水を飲む
```

```sentence-n
ほんをよむ
leo un libro
本を読む
```

## が -- sujeto / existencia

Marca quien hace la accion, o indica que algo existe.

```sentence-n
いぬがいる
hay un perro
犬がいる
```

```sentence-n
みずがつめたい
el agua esta fria
水が冷たい
```

## に -- destino / lugar / tiempo

Indica hacia donde, en donde (para existir), o cuando.

```sentence-n
がっこうにいく
voy a la escuela
学校に行く
```

```sentence-n
いえにねこがいる
hay un gato en la casa
家に猫がいる
```

## で -- medio / lugar de accion

Indica con que o donde se realiza una accion.

```sentence-n
にほんごではなす
hablo en japones
日本語で話す
```

```sentence-n
はしでたべる
como con palillos
箸で食べる
```

## の -- posesion

Conecta dos sustantivos: A の B = "B de A".

```sentence-n
ねこのなまえ
el nombre del gato
猫の名前
```

```sentence-n
にほんのやま
montañas de Japon
日本の山
```

# Ejercicios

## Completa con la particula correcta

1. 猫 \_\_ 水を飲む。 (el gato bebe agua)
2. 学校 \_\_ 行く。 (voy a la escuela)
3. 日本語 \_\_ 話す。 (hablo en japones)
4. 犬 \_\_ いる。 (hay un perro)
5. 猫 \_\_ 名前。 (el nombre del gato)
6. 本 \_\_ 読む。 (leo un libro)

---

## Respuestas

1. は, 2. に, 3. で, 4. が, 5. の, 6. を
