# 助詞 (じょし) - Particulas

Las particulas son palabras cortas que van despues de un sustantivo e indican su funcion
en la oracion. No tienen significado propio, solo conectan.

## は - tema

Se lee "wa" (no "ha") cuando es particula. Marca el tema de la oracion: "en cuanto a X..." Indica de que se esta hablando, no necesariamente quien hace la accion.

```sentence-n
ねこはどうぶつです
el gato es un animal
猫は動物です
```

```sentence-n
やまはたかいです
la montaña es alta
山は高いです
```

```sentence-n
せんせいはにほんじんです
el profesor es japones
先生は日本人です
```

Nota: は no dice "quien" hace algo, sino "de que" habla la oracion. Para marcar quien hace la accion se usa が.

## を - objeto directo

Se lee "o". Marca lo que recibe la accion del verbo -- el complemento directo. Responde a "que cosa?" despues del verbo.

```sentence-n
みずをのむ
bebo agua
水を飲む
```

```sentence-n
ほんをよむ
leo un libro
本を読む
```

```sentence-n
えいがをみる
veo una pelicula
映画を見る
```

## が - sujeto / existencia

Tiene dos usos principales.

### Sujeto

Marca quien hace la accion o quien tiene una cualidad. Se usa cuando la informacion es nueva o se responde a "quien?"

```sentence-n
ともだちがくる
mi amigo viene
友達が来る
```

```sentence-n
とりがとぶ
el pajaro vuela
鳥が飛ぶ
```

```sentence-n
みずがつめたい
el agua esta fria
水が冷たい
```

### Existencia

Indica que algo existe. Se usa con いる (seres vivos) y ある (objetos).

```sentence-n
いぬがいる
hay un perro
犬がいる
```

```sentence-n
みずがある
hay agua
水がある
```

```sentence-n
こうえんにはながある
hay flores en el parque
公園に花がある
```

## に - destino / lugar / tiempo

Tiene tres usos principales.

### Destino - hacia donde

```sentence-n
がっこうにいく
voy a la escuela
学校に行く
```

```sentence-n
にほんにいく
voy a Japon
日本に行く
```

```sentence-n
いえにかえる
regreso a casa
家に帰る
```

### Lugar de existencia - en donde hay algo

Se usa con いる y ある para indicar donde esta o existe algo.

```sentence-n
いえにねこがいる
hay un gato en la casa
家に猫がいる
```

```sentence-n
つくえのうえにほんがある
hay un libro sobre el escritorio
机の上に本がある
```

```sentence-n
にわにはながある
hay flores en el jardin
庭に花がある
```

### Tiempo - cuando

```sentence-n
しちじにおきる
me levanto a las siete
七時に起きる
```

```sentence-n
げつようびにあう
nos vemos el lunes
月曜日に会う
```

```sentence-n
さんがつににほんにいく
voy a Japon en marzo
三月に日本に行く
```

## で - medio / lugar de accion

Tiene dos usos principales.

### Medio / instrumento - con que

Indica la herramienta, el medio de transporte, o el idioma que se usa para hacer algo.

```sentence-n
はしでたべる
como con palillos
箸で食べる
```

```sentence-n
にほんごではなす
hablo en japones
日本語で話す
```

```sentence-n
バスでいく
voy en autobus
バスで行く
```

### Lugar de accion - donde se hace algo

Indica el lugar donde ocurre una accion activa (no solo existir).

```sentence-n
こうえんではしる
corro en el parque
公園で走る
```

```sentence-n
いえでべんきょうする
estudio en casa
家で勉強する
```

```sentence-n
レストランでたべる
como en el restaurante
レストランで食べる
```

Nota: に marca donde algo **existe** (estatico), で marca donde se **hace** algo (activo).
- {公園 (こうえん) に 花 (はな) がある} = hay flores en el parque (existencia)
- {公園 (こうえん) で 走 (はし) る} = corro en el parque (accion)

## の - posesion

Conecta dos sustantivos: A の B = "B de A". Indica que B pertenece a, o viene de, A.

```sentence-n
ねこのなまえ
el nombre del gato
猫の名前
```

```sentence-n
にほんのやま
montañas de Japon
日本の山
```

```sentence-n
せんせいのほん
el libro del profesor
先生の本
```

Se puede encadenar: {日本 (にほん) の 友達 (ともだち) の 名前 (なまえ)} = el nombre del amigo de Japon.

# Ejercicios

## Completa con la particula correcta

1. 猫 (ねこ) \_\_ 水 (みず) を 飲 (の) む。 (el gato bebe agua)
1. 学校 (がっこう) \_\_ 行 (い) く。 (voy a la escuela)
1. 日本語 (にほんご) \_\_ 話 (はな) す。 (hablo en japones)
1. 犬 (いぬ) \_\_ いる。 (hay un perro)
1. 猫 (ねこ) \_\_ 名前 (なまえ)。 (el nombre del gato)
1. 本 (ほん) \_\_ 読 (よ) む。 (leo un libro)
1. 友達 (ともだち) \_\_ 来 (く) る。 (mi amigo viene)
1. 公園 (こうえん) \_\_ 走 (はし) る。 (corro en el parque)
1. 先生 (せんせい) \_\_ 本 (ほん)。 (el libro del profesor)
1. 魚 (さかな) \_\_ 食 (た) べる。 (como pescado)
1. 花 (はな) \_\_ きれいです。 (la flor es bonita)
1. 東京 (とうきょう) \_\_ 住 (す) む。 (vivo en Tokio)
1. 電車 (でんしゃ) \_\_ 行 (い) く。 (voy en tren)
1. 鳥 (とり) \_\_ いる。 (hay un pajaro)
1. 母 (はは) \_\_ 料理 (りょうり)。 (la comida de mama)
1. 映画 (えいが) \_\_ 見 (み) る。 (veo una pelicula)
1. 日本 (にほん) \_\_ 行 (い) く。 (voy a Japon)
1. 山 (やま) \_\_ 大 (おお) きいです。 (la montaña es grande)
